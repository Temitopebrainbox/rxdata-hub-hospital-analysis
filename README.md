## 📌 Project Overview
This repository contains the Week 1 project for the **#BuildYourPortfolioWithMe** program at **RxData Hub**. 

In Data Analytics, a portfolio speaks much louder than a CV. This project focuses on cleaning messy real-world healthcare data, conducting exploratory data analysis (EDA), and deriving actionable key performance indicators (KPIs) to evaluate hospital operations.

---

## 📊 Dataset Snapshot & Key Metrics

* **Total Patients:** 10,000
* **Departments:** 5 main departments (with a granular breakdown of 252 department records)
* **Patient Types:** 3 categories
* **Average Waiting Time:** 75 minutes
* **Doctor's Score:** 2.8 / 5.0

### Dataset Fields Included:
* Count in the Hospital
* Visit Time (AM vs. PM)
* Patient Categories

> **Note:** The raw dataset required significant cleaning and structuring, making it an ideal real-world analytics project.

---

## 🛠️ Skills & Tools Applied
* **Data Cleaning & Preprocessing:** Handling raw, unstructured records.
* **Exploratory Data Analysis (EDA):** Uncovering trends across patient visits and departments.
* **KPI Derivation:** Extracting key hospital operational metrics.
* **Visualization & Dashboarding:** Structuring data for visual representation.
